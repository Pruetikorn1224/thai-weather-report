# Thai Water Report

วันที่อัพเดทล่าสุด: 2026-7-5

พยากรณ์อากาศ 24 ชั่วโมงข้างหน้า บริเวณภาคเหนือตอนบน ภาคตะวันออกเฉียงเหนือตอนบน และภาคตะวันออก มีฝนตกหนักมากบางพื้นที่ เนื่องจากร่องมรสุมพาดผ่านตอนบนของภาคเหนือ และประเทศลาวตอนบน เข้าสู่พายุโซนร้อนกำลังแรง “ไมสัก” (MAYSAK) ประกอบกับมรสุมตะวันตกเฉียงใต้ที่พัดปกคลุมทะเลอันดามัน ประเทศไทย และอ่าวไทยมีกำลังค่อนข้างแรง ขอให้ประชาชนระวังอันตรายจากฝนตกหนักถึงหนักมากและฝนที่ตกสะสม ซึ่งอาจทำให้เกิดน้ำท่วมฉับพลันและน้ำป่าไหลหลาก โดยเฉพาะพื้นที่ลาดเชิงเขาใกล้ทางน้ำไหลผ่านและพื้นที่ลุ่มไว้ด้วยสำหรับคลื่นลมที่พัดปกคลุมทะเลอันดามันตอนบนมีกำลังค่อนข้างแรง โดยมีคลื่นสูง 2-3 เมตร บริเวณที่มีฝนฟ้าคะนองคลื่นสูงมากกว่า 3 เมตร ส่วนทะเลอันดามันตอนล่างและอ่าวไทยตอนบนมีคลื่นสูงประมาณ 2 เมตร บริเวณที่มีฝนฟ้าคะนองคลื่นสูงมากกว่า 2 เมตร ขอให้ชาวเรือบริเวณทะเลอันดามันและอ่าวไทยเดินเรือด้วยความระมัดระวัง และหลีกเลี่ยงการเดินเรือในบริเวณที่มีฝนฟ้าคะนอง สำหรับเรือเล็กบริเวณทะเลอันดามันตอนบนควรงดออกจากฝั่งไว้ด้วย
