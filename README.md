# Thai Water Report

วันที่อัพเดทล่าสุด: 2026-7-20

พยากรณ์อากาศ 24 ชั่วโมงข้างหน้า ประเทศไทยมีฝนน้อย เนื่องจากมรสุมตะวันตกเฉียงใต้ที่พัดปกคลุมทะเลอันดามัน ประเทศไทย และอ่าวไทยมีกำลังอ่อน ขอให้ประชาชนระวังอันตรายจากฝนฟ้าคะนองไว้ด้วย  สำหรับคลื่นลมบริเวณทะเลอันดามันตอนบนและอ่าวไทยตอนบนมีกำลังปานกลาง โดยมีคลื่นสูง 1-2 เมตร ส่วนทะเลอันดามันตอนล่างมีคลื่นสูงประมาณ 1 เมตร บริเวณที่มีฝนฟ้าคะนองมีคลื่นสูงมากกว่า 2 เมตร ขอให้ชาวเรือเดินเรือด้วยความระมัดระวัง และหลีกเลี่ยงการเดินเรือในบริเวณที่มีฝนฟ้าคะนอง
