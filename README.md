# Thai Water Report

วันที่อัพเดทล่าสุด: 2026-7-12

พยากรณ์อากาศ 24 ชั่วโมงข้างหน้า ประเทศไทยมีฝนตกหนักบางแห่ง โดยมีฝนตกหนักมากบางพื้นที่ในภาคกลางด้านตะวันตก ภาคตะวันออก และภาคใต้ฝั่งตะวันตก เนื่องจากมรสุมตะวันตกเฉียงใต้ที่พัดปกคลุมทะเลอันดามัน ประเทศไทย และอ่าวไทยมีกำลังแรง ประกอบกับมีหย่อมความกดอากาศต่ำบริเวณประเทศเวียดนามตอนบน ขอให้ประชาชนในบริเวณดังกล่าวระวังอันตรายจากฝนตกหนักถึงหนักมากและฝนที่ตกสะสม ซึ่งอาจทำให้เกิดน้ำท่วมฉับพลันและน้ำป่าไหลหลาก โดยเฉพาะพื้นที่ลาดเชิงเขาใกล้ทางน้ำไหลผ่านและพื้นที่ลุ่มสำหรับคลื่นลมบริเวณทะเลอันดามันและอ่าวไทยมีกำลังแรงขึ้น โดยทะเลอันดามันตอนบนมีคลื่นสูง 2-4 เมตร บริเวณที่มีฝนฟ้าคะนองคลื่นสูงมากกว่า 4 เมตร ส่วนทะเลอันดามันตอนล่างและอ่าวไทยตอนบนมีคลื่นสูง 2-3 เมตร บริเวณที่มีฝนฟ้าคะนองจะมีคลื่นสูงมากกว่า 3 เมตร ขอให้ชาวเรือเดินเรือด้วยความระมัดระวัง และหลีกเลี่ยงการเดินเรือในบริเวณที่มีฝนฟ้าคะนอง เรือเล็กบริเวณทะเลอันดามันและอ่าวไทยตอนบนควรงดออกจากฝั่งจนถึงวันที่ 15 ก.ค. 69 ส่วนประชาชนบริเวณชายฝั่งภาคใต้ฝั่งตะวันตกควรระวังอันตรายจากคลื่นที่ซัดเข้าหาฝั่ง อนึ่ง พายุโซนร้อนกำลังแรง “บาหวี่” (BAVI) บริเวณมณฑลเจียงซี ประเทศจีนแล้ว คาดว่าจะอ่อนกำลังลงเป็นพายุโซนร้อน และพายุดีเปรสชันตามลำดับในระยะต่อไป โดยพายุนี้ไม่เคลื่อนเข้าสู่ประเทศไทย ขอให้ผู้ที่จะเดินทางไปในบริเวณดังกล่าวตรวจสอบสภาพอากาศก่อนออกเดินทางไว้ด้วย
