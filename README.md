# Thai Water Report

วันที่อัพเดทล่าสุด: 2026-7-2

พยากรณ์อากาศ 24 ชั่วโมงข้างหน้า ประเทศไทยมีฝนตกหนักบางแห่ง โดยมีฝนตกหนักมากบางพื้นที่ในภาคตะวันออก และภาคใต้ฝั่งตะวันตก เนื่องจากร่องมรสุมเลื่อนลงมาพาดผ่านภาคเหนือตอนบน และภาคตะวันออกเฉียงเหนือตอนบน ประกอบกับมรสุมตะวันตกเฉียงใต้ที่พัดปกคลุมทะเลอันดามัน ประเทศไทย และอ่าวไทยมีกำลังค่อนข้างแรง ขอให้ประชาชนระวังอันตรายจากฝนตกหนักถึงหนักมาก และฝนที่ตกสะสม ซึ่งอาจทำให้เกิดน้ำท่วมฉับพลันและน้ำป่าไหลหลาก โดยเฉพาะพื้นที่ลาดเชิงเขาใกล้ทางน้ำไหลผ่านและพื้นที่ลุ่มไว้ด้วยสำหรับคลื่นลมที่พัดปกคลุมทะเลอันดามันตอนบนมีกำลังค่อนข้างแรง โดยมีคลื่นสูง 2-3 เมตร บริเวณที่มีฝนฟ้าคะนองคลื่นสูงมากกว่า 3 เมตร ส่วนทะเลอันดามันตอนล่างและอ่าวไทยตอนบนมีคลื่นสูงประมาณ 2 เมตร บริเวณที่มีฝนฟ้าคะนองคลื่นสูงมากกว่า 2 เมตร ขอให้ชาวเรือบริเวณทะเลอันดามันและอ่าวไทยเดินเรือด้วยความระมัดระวัง และหลีกเลี่ยงการเดินเรือในบริเวณที่มีฝนฟ้าคะนอง สำหรับเรือเล็กบริเวณทะเลอันดามันตอนบนควรงดออกจากฝั่งไว้ด้วยอนึ่ง พายุดีเปรสชันบริเวณทะเลจีนใต้ตอนกลาง มีแนวโน้มจะทวีกำลังแรงขึ้น คาดว่าจะเคลื่อนผ่านเกาะไหหลำ และเคลื่อนขึ้นฝั่งประเทศจีนตอนใต้ในช่วงวันที่ 4-6 ก.ค. 69 โดยยังไม่ส่งผลกระทบโดยตรงต่อลักษณะอากาศของประเทศไทย
